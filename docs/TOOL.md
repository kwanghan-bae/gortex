# 🛠️ Gortex Custom Agent Tools

**Document Role**
이 문서는 에이전트가 개발 효율성을 높이기 위해 **직접 제작한 커스텀 도구(Scripts, Utilities)**의 카탈로그이다. 모든 도구는 제작 시 이곳에 명세가 기록되어야 하며, 이후의 에이전트들은 이 명세를 보고 도구를 활용할 수 있다.

---

## 1. Tool Creation Protocol

1.  **필요성 인지**: 기존 도구로 해결이 어렵거나 반복되는 작업이 발견될 때 도구 제작을 결정한다.
2.  **구현**: `gortex/scripts/` 또는 `gortex/utils/` 하위에 실행 가능한 형태로 구현한다.
3.  **명문화 (MANDATORY)**: 제작 즉시 이 파일(`TOOL.md`)에 아래 양식으로 명세를 추가한다.
    *   **도구명**: (파일명)
    *   **목적**: (왜 만들었는가)
    *   **사용법**: (실행 명령어 및 인자 설명)
    *   **입출력**: (Input/Output 형식)
